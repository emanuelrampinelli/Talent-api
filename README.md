## Instalação

1. Clone o repositório:

2. Instale as dependências com o Maven

3. Instale o [PostgresSQL](https://www.postgresql.org/)

5. Inicie a aplicação com o Maven

6. A API estará acessível em http://localhost:8080

## End-points da API
A API fornece os seguintes end-points:

```markdown
GET /product - Recupera uma lista de todos os produtos. (todos os usuários autenticados)

POST /product - Registra um novo produto (requer acesso de ADMIN).

```
